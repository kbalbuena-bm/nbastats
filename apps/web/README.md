# Web Frontend

This is the frontend web application for NBA Stats, built with Next.js.

## Environment Variables

Create a `.env.local` file in this directory (`apps/web/.env.local`) with:

```
NEXT_PUBLIC_API_URL=http://localhost:4000
```

The `NEXT_PUBLIC_` prefix makes this variable available in the browser.

## Available Commands

- `npm run dev` - Start development server (http://localhost:3000)
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run linter to check code quality
